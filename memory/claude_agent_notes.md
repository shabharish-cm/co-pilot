# Claude Agent Notes

Purpose: append-only notes for learnings, mistakes, and user preferences that should influence future behavior.

How to use:
- Add a note only when it is actionable and reusable.
- Keep each note short and specific.
- Include context and what to do next time.
- Do not rewrite old entries; append new dated entries.

## Entry Template

```md
## YYYY-MM-DD
- Type: [learning | mistake | preference]
- Context: <where this came up>
- Note: <what was learned / what went wrong / user preference>
- Action: <how Claude should behave next time>
```
